## 🐛 常见问题排查

### 问题1：图标不显示
**可能原因**：
- 图标文件路径错误
- 图标文件不存在
- 图标格式不支持

**解决方案**：
1. 检查文件路径是否与代码中一致
2. 确认图标文件已放置在正确目录
3. 确保图标格式为 PNG

### 问题2：图标显示模糊
**可能原因**：
- 图标尺寸过小
- 图标未提供多倍率版本
- 图标压缩过度

**解决方案**：
1. 使用更高分辨率的图标
2. 提供 @2x、@3x 版本
3. 使用无损压缩

### 问题3：图标颜色不对
**可能原因**：
- 图标本身颜色不符合设计规范
- CSS 滤镜影响了图标颜色

**解决方案**：
1. 重新设计图标，使用正确的颜色
2. 检查 CSS 中是否有 filter 属性

### 问题4：TabBar 图标不显示
**可能原因**：
- pages.json 配置错误
- 图标尺寸不符合要求（应为 81x81px）

**解决方案**：
1. 检查 pages.json 中的 iconPath 和 selectedIconPath
2. 确保图标尺寸为 81x81px
3. 重新编译项目

## 📝 图标使用示例

### 动态切换图标
```vue
<template>
  <image 
    class="icon" 
    :src="batteryIcon" 
    mode="aspectFit"
  />
</template>

<script>
export default {
  computed: {
    batteryIcon() {
      const battery = this.battery
      if (battery > 80) return '/static/icons/battery-full.png'
      if (battery > 50) return '/static/icons/battery-medium.png'
      if (battery > 20) return '/static/icons/battery-low.png'
      return '/static/icons/battery-critical.png'
    }
  }
}
</script>
```

### 条件渲染图标
```vue
<template>
  <image 
    class="layer-icon" 
    :src="currentLayer === 'dark-vector' ? '/static/icons/satellite.png' : '/static/icons/terrain.png'" 
    mode="aspectFit"
  />
</template>
```
