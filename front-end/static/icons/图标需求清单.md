## 💡 实现方案

### 方案1：使用 Emoji（推荐）✅

**优点**：
- 无需准备图片文件
- 跨平台兼容性好
- 文件体积小
- 色彩丰富

**示例代码**：
```vue
<view class="medal-item">
  <text class="medal-icon">🦅</text>
  <text class="medal-name">猎鹰勋章</text>
</view>
```

---

### 方案2：使用 uni-icons 组件 ✅

**优点**：
- uni-app 内置组件
- 图标库丰富
- 支持自定义颜色和大小
- 性能好

**示例代码**：
```vue
<uni-icons type="refreshempty" size="30" color="#73D13D"></uni-icons>
<uni-icons type="download" size="30" color="#00d4ff"></uni-icons>
```

**常用图标映射**：
- 刷新：`refreshempty`
- 下载：`download`
- 删除：`trash`
- 设置：`gear`
- 锁：`locked`

---

### 方案3：使用 CSS 绘制 ✅

**优点**：
- 完全自定义
- 无需额外文件
- 支持动画效果

**示例代码**：
```scss
// 刷新图标
.refresh-icon {
  width: 40rpx;
  height: 40rpx;
  border: 3px solid #73D13D;
  border-top-color: transparent;
  border-radius: 50%;
  animation: rotate 1s linear infinite;
}

@keyframes rotate {
  to { transform: rotate(360deg); }
}

// 信号强度图标
.signal-icon {
  display: flex;
  align-items: flex-end;
  gap: 4rpx;
  height: 40rpx;
  
  .bar {
    width: 8rpx;
    background: #73D13D;
    border-radius: 2rpx;
    
    &:nth-child(1) { height: 25%; }
    &:nth-child(2) { height: 50%; }
    &:nth-child(3) { height: 75%; }
    &:nth-child(4) { height: 100%; }
  }
}
```

## 📝 使用建议

1. **优先使用已有图标**
   - 检查 `/static/icons/`、`/static/icons-2/`、`/static/icons-3/` 是否有可复用的图标

2. **其次使用 Emoji**
   - 适用于装饰性图标（勋章、状态等）
   - 跨平台兼容性好

3. **再次使用 uni-icons**
   - 适用于通用功能图标（刷新、下载、设置等）
   - 性能好，易于维护

4. **最后使用 CSS 绘制**
   - 适用于简单几何图形（圆形、方形、线条等）
   - 支持动画效果
