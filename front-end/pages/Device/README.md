# 设备页说明

## 页面定位

“设备”页用于展示边境防控设备的整体运行状态，帮助值守人员快速判断：

- 当前设备网络是否健康
- 哪些设备需要优先处理
- 离线地图与缓存资源是否需要管理
- 哪些态势点位需要联动关注

页面重点不是单纯“列设备”，而是把系统状态、设备状态和现场处置建议放在同一视图里。

---

## 页面结构

### 1. 顶部系统状态区
包含两部分：

- **系统健康指数**：按设备状态、电量、信号综合计算
- **同步状态条**：展示当前数据同步、离线缓存与上传情况

### 2. 中部设备拓扑区
用于展示：

- 广西边境重点区域设备分布
- 边缘节点与设备连线关系
- 在线、预警、离线状态差异
- 与 GIS 态势数据的叠加显示

### 3. 下部设备与资源管理区
用于展示：

- 设备卡片列表
- 设备电量与信号强度
- 离线地图包状态
- 已上传缓存清理入口

## 核心能力

### 系统健康计算
通过 `diagnosticLogic.js` 计算系统健康分数，主要参考：

- 在线 / 预警 / 离线状态
- 电量水平
- 信号强度

### 设备优先级排序
设备列表按优先级排序：

1. 离线设备
2. 预警设备
3. 在线设备
4. 同状态下低电量优先

### 设备详情查看
点击地图标记或设备卡片后，可查看：

- 设备 ID
- 设备类别
- 厂家与型号
- 电量、信号、最近活跃时间
- 最近触发事件
- 远程控制或创建检查任务入口

### 离线资源管理
支持：

- 离线地图包下载
- 已下载地图包删除
- 采样缓存清理
- 存储占用展示

## 交互说明

### 刷新设备状态
- 点击右上角刷新按钮可立即刷新
- 页面会定时自动刷新设备状态

### 查看地图标记
- 点击设备标记可打开对应设备详情
- 点击 GIS 态势标记可联动查看相关点位信息

### 下载地图包
- 点击“下载”后显示进度
- 下载完成后显示“已下载”状态

### 清理缓存
- 点击“清理已上传缓存”后弹出确认框
- 清理完成后同步更新存储占用

## 视觉说明

设备页整体采用深色指挥终端风格，重点强化三类反馈：

- **绿色**：在线 / 正常
- **橙色**：预警 / 同步中
- **灰色或红色**：离线 / 高风险

同时通过光环、脉冲、流动线条等方式增强“系统仍在运转”的感知。
