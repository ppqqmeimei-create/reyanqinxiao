# 前端项目 (Front-end)

这是热眼擒枭项目的前端部分。

## 项目结构

```
front-end/
├── pages/              # 页面文件
├── components/         # 组件文件
├── static/            # 静态资源
├── utils/             # 工具函数
├── App.vue            # 主应用文件
├── main.js            # 入口文件
├── pages.json         # 页面配置
└── manifest.json      # 应用配置
```

## 功能模块

1. **GIS地图** - 污染源追溯和扩散预测
2. **预警工作台** - 多源预警整合和智能派警
3. **检查任务** - 智能检查清单和采样
4. **设备管理** - 企业风险画像
5. **数据大屏** - 实时监控展示
6. **个人中心** - 用户信息管理

## 技术栈

- Vue 3
- uni-app
- SCSS
- JavaScript
