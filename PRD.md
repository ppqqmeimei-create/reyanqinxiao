## 一、产品定位与目标

### 1.1 产品名称

**热眼擒枭 —— 边境活物走私智能防控引领者**

### 1.2 产品定位

热眼擒枭是一款专为边境一线执法人员打造的**边境活物走私智能防控平台**。项目聚焦广西边境口岸等复杂地带，依托多模态感知技术与AI算法，实现对穿山甲、象牙、食蟹猴、海龟等濒危物种走私行为的精准识别与实时预警。

产品核心价值主张：**"热眼"识万物，"擒枭"守边境** —— 利用人工智能和多模态感知技术，打造一双洞察一切的"热眼"，精准识别走私行为，让濒危物种走私无处遁形。以技术创新为驱动，成为边境活物走私智能防控领域的**引领者**。

### 1.3 产品目标

#### 核心业务目标

| 目标维度 | 具体指标 |
|---------|---------|
| 识别准确率 | 多传感器融合综合识别准确率 ≥ 85%，核心预警准确率 94.5% |
| 响应时效 | 预警响应时间 ≤ 1 分钟（传统方案 30 分钟以上） |
| 夜间识别 | 支持全天候 24 小时红外热成像识别 |
| 执法效率 | 预警处置率 ≥ 87%，任务完成率 ≥ 92% |
| 一线可用性 | 戴手套盲操成功率 ≥ 90% |
| 打击成效 | 年侦办穿山甲等濒危物种走私案件 ≥ 12起 |

#### 技术架构目标

| 目标维度 | 具体描述 |
|---------|---------|
| 云-边-端协同 | 边缘计算降低带宽成本，毫秒级响应 |
| 离线韧性 | 弱网/断网环境下仍可完成核心业务操作 |
| 防伪溯源 | 证据链不可篡改，支持一键校验 |
| 全流程闭环 | 感知→预警→处置→取证→研判 完整数字化闭环 |

### 1.4 差异化优势

| 维度 | 传统方案 | 热眼擒枭 |
|-----|---------|---------|
| 识别准确率 | 65% | 94.5% |
| 响应时间 | 30分钟+ | 1分钟 |
| 夜间识别 | 不支持 | 全天候支持 |
| 离线工作 | 不支持 | 完整支持 |
| 证据溯源 | 无 | 防伪校验体系 |

## 三、核心功能需求

### 3.1 功能模块总览

- **认证授权模块**：统一身份认证（JWT）、RBAC角色权限控制、离线Token缓存、敏感操作二次验证
- **态势感知模块（GIS）**：五大战区实时态势图、多图层可视化（红外/振动/巡逻路线/走私热区）、影子追踪与轨迹预测、拦截点智能计算
- **预警中心模块**：三级预警体系（紧急/警告/提示）、濒危物种识别、智能分配与联动处置、多条件过滤与去重合并、预警趋势统计分析
- **任务调度模块**：四类任务（快速勘查/完整取证/定点检查/持续巡逻）、任务创建/分配/执行/验收全流程、进度实时跟踪、任务关联预警与取证
- **设备管理模块**：多种传感器设备监控、设备状态评估与预警、灵敏度远程配置、设备健康分计算
- **执法取证模块**：拍照/录像取证、自动水印叠加、防伪校验码生成、离线缓存与断点续传、证据链一键导出
- **情报研判模块**：多维度数据统计、濒危物种分析、关系图谱可视化、走私热点聚合
- **个人中心模块**：个人信息管理、战绩统计、同步中心、操作模式设置

### 3.2 核心业务流程

#### 预警处置闭环

1. 传感器触发（红外/振动/可见光检测到可疑目标）
2. 多传感器时空融合（5分钟窗口+500米空间）
3. AI濒危物种识别 + 置信度评分（0~100）
4. 风险等级判定（critical ≥ 85 / warning ≥ 70 / info < 70）
5. 紧急预警自动推送 → 指挥中心确认
6. 任务创建 → 分配执勤人员
7. 一线执行处置 → 拍照/录像取证
8. 任务完成 → 证据上传
9. 预警关闭 → 案件归档

#### 离线取证流程

1. 一线执勤 → 发现可疑目标/收到走私预警
2. 判断网络状态（在线直接上传 / 离线本地缓存证据）
3. 网络恢复 → 自动同步上传
4. 服务器校验 → 证据入库

### 8.1 短期扩展（3-6个月）

- **智能巡逻路径规划**：基于走私热点自动规划最优巡逻路径，提升巡逻效率30%
- **多语言支持**：支持越南语、老挝语等边境国家语言，跨境协作语言支持
- **濒危物种库扩展**：扩展AI识别物种库至CITES附录物种，识别准确率≥95%

### 8.2 中期扩展（6-12个月）

- **无人机自主巡逻**：无人机按预设航线自动巡逻，实时回传视频流，适用于山区、林区等人员难以到达区域
- **区块链存证**：证据数据上链存储，确保不可篡改，提升证据法律效力
- **AR辅助执法**：AR眼镜显示实时预警信息、导航指引，解放双手，提升执法效率
- **多部门协同平台**：接入海关、林业、边检等部门数据，形成联防联控机制

### 8.3 长期扩展（1-3年）

- **跨境协作网络**：与越南、老挝、缅甸等国边境管理部门建立数据共享，服务东盟国家
- **走私风险预测**：基于历史数据预测走私风险趋势和热点区域，准确率≥80%
- **智能穿戴集成**：智能手表/战术耳机集成，提升执法人员安全保障
- **卫星遥感融合**：接入卫星遥感数据，扩大监控覆盖面，实现全天候全域监控

### 8.4 技术演进路线

- **v3.0 (当前)**：纯走私防控版本，移除所有环境监测内容
- **v3.1**：濒危物种库扩展 + 巡逻路径规划
- **v3.5**：无人机集成 + 区块链存证
- **v4.0**：AR辅助执法 + 多部门协同
- **v4.5**：跨境协作 + 走私风险预测
